

Nas-FiftyK aims to be a tool that allow to store small binary files into the Nebulas Blockchain.
The maximum file size allowed is 50 kilobytes but we think that it is sufficient to increase creativity as Twitter performed it with 280 characters limit size. 

Example of use cases:
    - store and share public files with others like avatars
    - increase Dapps features by storing binary content
    - secure your private data, accessible anywhere, by storing your files with a very strong password encrypted password
    - ...
